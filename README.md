# jagansharma.com

Official repository for **jagansharma.com** — a personal developer hub featuring interactive tools, experiments, and documentation.

The site is designed to function as a **coding playground and development showcase**, combining learning resources, live code tools, and experimental web projects.

---

## Overview

This project powers the website:

```
https://jagansharma.com
```

The goal of the site is to create a **centralized development hub** that includes:

* Interactive coding tools
* Developer documentation
* Project showcases
* Experimental web applications
* Learning resources for web development

---

## Features

### Developer Playground

A browser-based coding playground that allows users to:

* Write **HTML**
* Write **CSS**
* Write **JavaScript**
* Run code instantly in a live preview
* Generate **shareable playground links**

The playground uses the editor engine behind **Visual Studio Code**.

---

### Interactive Documentation

The documentation system includes:

* deep-dive explanations of project components
* code examples and architecture notes
* expandable sections for easy navigation
* developer-focused design patterns

---

### Project Showcase

The site also hosts demonstrations and prototypes including:

* web experiments
* developer utilities
* interactive UI components
* programming tools

## Development

Clone the repository:

```
git clone https://github.com/USERNAME/jagansharma.com
```

Run locally:

```
open index.html
```

or serve with a local dev server:

```
python3 -m http.server
```
