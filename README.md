# Front Desk AI Demo

This project is a native macOS SwiftUI demo for an AI receptionist product aimed at service businesses like dental clinics, med spas, and home services teams.

## What It Shows

- Revenue recovery from missed calls and texts
- Lead triage and booking-ready inbox workflows
- Appointment schedule visibility
- Follow-up automation monitoring

## Run

```bash
swift run UnoNotch
```

## Notes

- The executable target is still named `UnoNotch`, but the in-app product is presented as `Front Desk AI`.
- The current experience is powered by local sample data so it opens immediately without external services.
